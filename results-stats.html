WPE Release Test Results

This page processes the json output from https://results.webkit.org/, focusing on the results from the WPE-Linux-64-bit-Release-Tests post-commit bot.

This bot is defined by the following configuration object in the results server API:

  • Platform: WPE
  • Style: release
  • Flavor: wk2 (There's another flavor for the legacy WPE API bot since mid November 2025).

      • To work around some CORS limitations querying the API, we're using a proxy script at https://people.igalia.com/clopez/results_webkit_proxy_json.php that accepts the already mentioned platform, release, flavor, and limit parameters.

      To see the custom results, save this link to a file and use the dialog below to load the saved file. Feel free to adjust the parameters according to the results.webkit.org documentation, specially the query parameters.

      Note that currently we don't have a direct way to map a result entry to the build job that created it, so you might need to correlate through the timestamp (UTC).
